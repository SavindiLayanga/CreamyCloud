# Creamy Delights

A modern **Point of Sale (POS) system** for an ice cream shop, built using **HTML, CSS, Tailwind CSS, and Bootstrap**.
This system provides a simple and user-friendly interface to manage orders, products, and billing.

---

## 🚀 Features

* 🛒 Add ice cream items to cart
* 💵 Real-time price calculation
* 🧾 Generate order summary
* 🔍 Search and filter products
* 📱 Responsive design (mobile + desktop)
* 🎨 Clean and modern UI

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the application
* **CSS3** – Custom styling
* **Tailwind CSS** – Utility-first styling
* **Bootstrap** – Components and layout
